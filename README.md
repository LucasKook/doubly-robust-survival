# Comparison of doubly robust conditional independence tests for survival responses

The code in `./code/comparison.R` compares a model-X based conditional
independence test proposed in [1] with the TRAM-GCM test proposed in [2] and
implemented in the [`comets` R
package](https://github.com/LucasKook/comets.git) [3].
